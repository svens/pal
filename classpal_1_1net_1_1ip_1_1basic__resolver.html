Detailed Description

template<typename InternetProtocol>
class pal::net::ip::basic_resolver< InternetProtocol >

basic_resolver provides name resolution service.

Name resolution is the translation of a host and service name into a sequence of endpoints, or the translation of endpoint into it's corresponding host and service name.
