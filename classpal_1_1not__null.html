Detailed Description

template<typename Pointer>
class pal::not_null< Pointer >

Fancy pointer that restricts internal plain or smart Pointer to hold only non-null values.

It has zero overhead over Pointer.

Return internal pointer as movable and nullable pointer type allowing to take ownership of pointed element.

If ownership is moved out of this, it remains undefined state (i.e. can be actually null).
