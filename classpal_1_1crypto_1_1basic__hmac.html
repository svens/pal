Detailed Description

template<typename Algorithm>
class pal::crypto::basic_hmac< Algorithm >

One-way cryptographic keyed hashing using Algorithm.

If all data is already fully present, use static one_shot() to calculate digest immediately.

If data is not completely present, instantiate this class and add new data in one or more calls to update(). Once full data has been fed into hasher, call finish() to fetch calculated digest. Application can immediately reuse instance after call to finish(), it is re-initialized internally.
