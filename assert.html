#define pal_assert (   Expr,
  OptionalMessage 
)    /* internal */

Require Expr to evaluate to true.

On success, returns result of Expr. Otherwise, throw std::logic_error. If OptionalMessage is specified, it is passed to std::logic_error constructor as parameter. If no OptionalMessage, textual representation of Expr with source code location information is used.

Note
It is similar to standard assert() macro, except Expr is always evaluated and result is returned.
