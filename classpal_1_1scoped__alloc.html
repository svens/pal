Detailed Description

template<typename T, size_t StackSize>
class pal::scoped_alloc< T, StackSize >

Scoped allocation: up to StackSize elements of T are allocated from stack.

If actually requested size is bigger (specified during construction), memory is allocated from heap instead.

If memory is taken from heap, it is automatically freed during destruction.
